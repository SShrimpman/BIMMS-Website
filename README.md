## To Initialize:

- git init (on folder you want to work)
- git clone githubLinkOfThisRepository.com

What is needed:

- Node.js
- PHP v.8.2.4
- Composer

To run the project:

- npm install
- npm run dev
- composer install
- php artisan migrate (after changing the name of the database on .env file)
- php artisan serve
